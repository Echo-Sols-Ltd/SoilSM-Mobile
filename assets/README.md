# Assets Directory

This directory contains app assets required by Expo.

## Required Assets

You need to create the following image files:

1. **icon.png** - App icon (1024x1024px)
   - Used as the app icon on home screen
   - Should be square with no transparency

2. **splash.png** - Splash screen image (1284x2778px recommended)
   - Shown when app is launching
   - Should match your app's branding

3. **adaptive-icon.png** - Android adaptive icon (1024x1024px)
   - Used for Android adaptive icons
   - Should have safe area (inner 66% of image)

4. **favicon.png** - Web favicon (48x48px)
   - Used when running on web
   - Small icon for browser tab

## Quick Setup

For now, you can use placeholder images or create simple colored squares:

1. Create a 1024x1024px green square for `icon.png` (use your brand color #4CAF50)
2. Create a 1284x2778px image with your logo for `splash.png`
3. Copy `icon.png` to `adaptive-icon.png`
4. Create a 48x48px version for `favicon.png`

## Online Tools

You can use these tools to create the assets:
- [App Icon Generator](https://www.appicon.co/)
- [Icon Kitchen](https://icon.kitchen/)
- [Favicon Generator](https://favicon.io/)

## Temporary Solution

If you just want to test, you can:
1. Use any 1024x1024px image as `icon.png`
2. Copy it to other required files
3. The app will work, just with placeholder images
